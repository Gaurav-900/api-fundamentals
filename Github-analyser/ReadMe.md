# API Fundamentals

Learning JavaScript API integration from scratch.

## What each file demonstrates

**01-basic-fetch.js**
Async/await, error handling, parsing JSON responses

**02-promise-all-filter.js**  
Parallel API calls with Promise.all, data filtering, 
transforming raw responses into clean output

**03-github-analyser.js**
Real API authentication with Bearer tokens, query 
parameters, combining multiple endpoints into one output

## How to run
1. Clone the repo
2. npm install
3. Copy .env.example to .env and add your GitHub token
4. node 03-github-analyser.js
